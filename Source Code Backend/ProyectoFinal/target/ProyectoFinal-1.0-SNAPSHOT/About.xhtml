Misión

La Universidad es una etapa importante en la vida estudiantil, pero puede ser muy estresante, enfocarse en el estudio, obtener buenas calificaciones, todo ello mientras continúas trabajando en actividades extracurriculares, teniendo una vida social y planificando para el futuro. El esfuerzo continuo por mantenerse en un nivel de excelencia puede ser agotador para ambos estudiantes y educadores.

Objectivos

El objetivo de este proyecto es construir una biblioteca en línea, con recursos de aprendizaje para materias específicas dirigido a una comunidad de estudiantes y/o educadores dedicados al aprendizaje colaborativo, tanto dentro como fuera del aula.
